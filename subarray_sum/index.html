Subarray Sum = k

How It Works

This algorithm finds the count of subarrays summing to k in O(n) time and O(n) space.

Key Insight

Let the prefix sum ending at index i be prefix_sum. If there exists an index j (where j < i) with a prefix sum equal to prefix_sum - k, then the subarray from j + 1 to i must sum exactly to k. We use a frequency ledger (hash map) to remember how many times we've seen each prefix sum in the past.

Algorithm Steps

  1. 1 Initialize prefix_sum = 0, count = 0, and prefix_count = { 0: 1 } (empty prefix).
  2. 2 For each number, add it to prefix_sum.
  3. 3 Check the ledger: if prefix_sum - k is present, add its frequency to our total count.
  4. 4 Increment the frequency of prefix_sum in the ledger by 1.

Complexity

Time: O(n) — single pass over the array.

Space: O(n) — to store prefix sum frequencies in the ledger.
