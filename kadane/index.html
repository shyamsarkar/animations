Kadane's Algorithm

Find the maximum sum of a contiguous subarray in linear time.

Array
How It Works

Kadane's algorithm efficiently finds the maximum sum of any contiguous subarray in a single pass through the array.

Key Insight

At each position, we decide: should we extend the current subarray, or start a new one? We start a new subarray if the current sum becomes negative, since adding a negative number would only decrease our total.

Algorithm Steps

  1. 1 Initialize current_sum and max_sum to the first element.
  2. 2 For each element, add it to current_sum.
  3. 3 If current_sum becomes negative, start a new subarray from the current element.
  4. 4 Update max_sum whenever current_sum is larger.

Complexity

Time: O(n) — single pass through the array.

Space: O(1) — only a few variables are tracked.
